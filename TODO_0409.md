## 2. 신청해야 할 API: **Research API**

제3자의 TikTok 채널/유저 데이터를 조회하려면 **Research API**가 필요함.

| API 제품 | 용도 | 비고 |
|---------|------|------|
| **Research API** | 타인의 유저 프로필/통계/영상 조회 | **이거 신청해야 함** |
| Login Kit | 본인 계정 정보만 조회 가능 | OAuth 동의 필요, 부적합 |
| Business API | 광고/비즈니스용 | 부적합 |

### Research API로 조회 가능한 데이터
- 유저 프로필 (display name, bio, avatar, 인증 여부)
- 팔로워/팔로잉/좋아요 수 통계
- 영상 검색 (해시태그, 키워드, 날짜 범위)
- 영상 댓글 데이터

### 주요 엔드포인트
```
POST /v2/research/user/info/       → 유저 프로필 조회
POST /v2/research/video/query/     → 영상 검색/조회
POST /v2/research/video/comments/  → 댓글 조회
```

---

## 3. 신청 절차 (Step by Step)

1. **개발자 계정 생성**: https://developers.tiktok.com/ 에서 TikTok 계정으로 로그인 → 개발자 등록
2. **앱(App) 생성**: Developer Portal에서 새 앱 생성
3. **Research API 제품 추가**: 앱 설정 → "Add products" → **"Research API"** 선택
4. **신청서 작성**: 아래 정보 제출
   - 소속 기관 정보 (회사, 연구소 등)
   - 데이터 사용 목적 및 계획 상세 설명
   - 데이터 보관/보안 계획
5. **심사 대기**: 수일~수주 소요
6. **승인 후**: Client Key + Client Secret 발급 → API 호출 가능

---

## 4. 기술 스펙

- **인증 방식**: OAuth 2.0 (Client Credentials Grant)
- **Token endpoint**: `https://open.tiktokapis.com/v2/oauth/token/`
- **API Base URL**: `https://open.tiktokapis.com/v2/`
- **데이터 형식**: JSON 기반 REST API

---

## 5. 비용 및 제한

- **비용**: 무료 (승인된 사용 목적 한정)
- **Rate Limit**: 일일 약 1,000 requests/day (승인 시 할당량 확정)
- 할당량 증가 필요 시 TikTok에 별도 요청 가능

---

## 6. 주의사항

- 연구/비즈니스 목적을 명확히 기술해야 승인 가능성 높음
- 개인 신청은 승인이 어려울 수 있음 → 소속 기관 명의 권장
- TikTok API 정책은 자주 변경되므로 최신 문서 반드시 확인할 것
